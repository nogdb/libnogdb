# Figma MCP Integration

Connect Figma designs directly to your development workflow.

## Setup

```bash
# 1. Install Figma MCP Server
npm install -g figma-mcp-server

# 2. Set environment variables
export FIGMA_TOKEN=your_figma_access_token
export FIGMA_TEAM_ID=your_team_id

# 3. Restart Windsurf to load MCP server
```

## Commands

### List Designs
```bash
figma-mcp list-designs
```

### Get Design Details
```bash
figma-mcp get-design --file-id="figma_file_id"
```

### Extract Design Tokens
```bash
figma-mcp extract-tokens --file-id="figma_file_id"
```

Output:
```json
{
  "colors": {
    "primary": "#3B82F6",
    "secondary": "#10B981",
    "accent": "#F59E0B"
  },
  "typography": {
    "fontFamily": "Inter, sans-serif",
    "fontSize": {
      "sm": "0.875rem",
      "base": "1rem",
      "lg": "1.125rem"
    }
  },
  "spacing": {
    "sm": "0.5rem",
    "md": "1rem",
    "lg": "1.5rem"
  }
}
```

### Export Components
```bash
figma-mcp export-components --file-id="figma_file_id"
```

### Validate Implementation
```bash
figma-mcp validate --component="Button" --design="figma_file_id"
```

## Design-to-Code Workflow

### Phase 1: Designer Creates Designs
- Create designs in Figma with proper component structure
- Establish design system (colors, typography, spacing)
- Share designs with development team

### Phase 2: AI Agents Analyze
- Design Agent reads Figma files via MCP
- Extracts design tokens automatically
- Analyzes component structure

### Phase 3: Developers Implement
- Cascade generates components based on designs
- Design Agent applies design tokens
- Frontend Dev refines implementation

### Phase 4: Validate
- Design Agent validates against original designs
- Reports discrepancies
- Suggests fixes

## Generated Component Example

```typescript
// Generated by Design Agent + Cascade
import { designTokens } from './design-tokens';

export const Button = ({ variant = 'primary', children }) => {
  const styles = {
    primary: {
      backgroundColor: designTokens.colors.primary,
      color: 'white',
      padding: `${designTokens.spacing.sm} ${designTokens.spacing.md}`,
      borderRadius: '8px',
      fontFamily: designTokens.typography.fontFamily
    }
  };

  return <button style={styles[variant]}>{children}</button>;
};
```
